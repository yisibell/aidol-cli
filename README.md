# aidol-cli
A command line tool for quickly building blog sites.

# Installation

``` bash
npm i @aidol/cli -g
```

# Usage

``` bash
$ aidol init <projectName>

$ cd <projectName>

$ yarn install

$ yarn dev
```

Or, just run `npx @aidol/cli init <projectName>` if you do not want pre-install `aidol` command globaly.
